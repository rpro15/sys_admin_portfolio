# Interview Self-Pitch (30-45 seconds)

Hello. I built this practical Home-Lab to demonstrate real system administration skills on tasks close to production support.

In this lab, I deployed core domain services (AD DS, DNS, DHCP), configured network segmentation with VLAN 10/20, and automated routine operations using PowerShell and Ansible.

I also implemented bulk user provisioning from CSV with home folders and access permissions, plus a nightly file server backup workflow with a documented test restore scenario.

The repository contains not only scripts and configs but also operational artifacts: a ticket template, a field-visit report example, a network diagram, and a short demo scenario.

In short, I can not only configure infrastructure, but also bring solutions to a working, verifiable, and documented state.
